# PC Care Utility

This is a safe maintenance free, bloatware free, and safe to use PC clean up utility.

**TESTED ONLY ON WINDOWS 10/11!!**


## Program Features

* **System Cleanup**: Empties/Clears: Temp files, Recycle Bin, Downloads folder
* **Browser Cache Cleaner**: Chrome, Edge, Firefox, Brave (cache only - never history/passwords)
* **Disk Space Analyzer**: Visual folder size breakdown
* **Power Management**: Change plans, reboot, shutdown
* **Network Tools**: Reset configuration, flush DNS, renew IP
* **Startup Manager**: View programs (safe read-only mode)
* **System Restore Point Creator**: Safety net before cleanup operations
* **Privacy Guard**: Disables Windows Data tracking usage and deletes bloatware.

## Safety First

* **NO registry cleaners** (modern Windows doesn't need them)
* **NO BIOS/UEFI operations** (physically dangerous)
* **All destructive operations require explicit confirmation**
* **Never deletes browser history/passwords/bookmarks**
* **Requires Administrator rights** (for legitimate system access)

## Usage

1. Please ensure that you run the program as administrator as it requires elevated permissions for some operations.
2. Upon the first time this software is opened, SmartScreen might give a warning. To bypass this, click on \*MORE INFO\* and then \*RUN ANYWAY\*.
3. Select your desired operation from the main menu
4. Confirm destructive operations when prompted
